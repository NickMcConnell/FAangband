# File:  themed.txt
#
# Function:  Supplies the game with information needed to create themed 
# dungeon levels.

# Since each themed level is defined grid-by-grid, each template takes 
# (approximately) 14000 characters.  This means there can't be very 
# many of them!  Therefore, themed levels are generated /very/ rarely, 
# and each can only appear once per character lifetime.

# Unless you need to fix a problem, I would suggest not studying these 
# levels.  They are designed to come as a sudden surprise.

# All themed levels rely on at least some internal code to maintain 
# consistency.  All lower the distance at which you and monsters can see 
# each other, do not add new random monsters to the level, amd restrict 
# the kinds of monsters that can appear initially.

# There are also restrictions as to where on the world map these levels can
# appear; these are currently hard-coded.

#name: name
#message: Message that replaces the normal level feeling
#D: A square-by-square description of the level.
